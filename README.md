# ハノイの塔

## ルール

三本の柱がある
leftに円盤がX枚積みあがっている
leftに積みあがっている円盤をすべてrightに移動させる
必ず一枚ずつしか移動できない
移動する円盤は自分より大きい円盤の上には載せられない

一番大きい円盤をXとする<br>
偶数、奇数時で解き方が違う解など様々な解があるが今回は同じ作業を繰り返す解を利用する<br>

---

一枚の場合

1:Xをrightへ

end

---

二枚の場合

1:X-1をcenterへ
2:Xをrightへ
3:X-1をrightへ

end

---

三枚の場合

1:X-2をrightへ
2:X-1をcenterへ
3:


---

考え

動かす円盤の大きさ、どこから動かすか、どこに動かすか、作業用の場所が必要<br>
size, from, target, work<br>
大きい円盤から動かすようにすれば解けそうです
